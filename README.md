<h1> Installation </h1>
<p> Option 1. <b> via Curseforge </b> - Download and import The Yuri Soup NetherUpdate.zip </p>
<p> Option 2. <b> via Modrinth </b> - Download and import The Yuri Soup NetherUpdate Md.mrpack </p>
<h1> Connecting to the server </h1>
<p> 1. Run minecraft WITH the modpack </p>
<p> 2. Go to MULTIPLAYER </p>
<p> 3. Connect to mc.yurisoup.online server </p>
<p> 4. Register by using /register [urpassword] [urpassword]</p>
